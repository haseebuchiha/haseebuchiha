I build applied-AI systems end to end — **Retrieval-Augmented Generation**, **multi-agent orchestration**, and **LLM-powered products**. Python/FastAPI on the backend, Next.js/React on the front, with real depth in vector + knowledge-graph retrieval and self-hosted local inference. 6+ years shipping production systems, on a foundation of Ruby on Rails and PostgreSQL.

> Outside client work I run a self-hosted **AI Factory** at home — a little crew of local LLM agents I'm forever breaking and rebuilding.
